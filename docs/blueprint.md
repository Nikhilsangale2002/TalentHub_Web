# **App Name**: TalentHub

## Core Features:

- Job Posting Display: Display job postings fetched from the Flask backend API, formatted for easy browsing.
- User Authentication: Secure user authentication via the Auth Service in the Flask backend.
- Resume Upload: Allow users to upload resumes that are sent to the Resume Service in the Flask backend for processing.
- AI-Powered Job Scoring: Use the AI Scoring Service to score and rank job applications, incorporating tools to assess candidate fit based on skills and experience.
- Search and Filter: Implement search and filter functionalities to allow users to find suitable jobs quickly.
- Application Tracking: Allow users to track the status of their job applications.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to evoke trust and professionalism.
- Background color: Very light blue (#E3F2FD), near white, for a clean and modern feel.
- Accent color: A gentle teal (#4DB6AC) for highlighting important actions.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, 'Inter' (sans-serif) for body text. Space Grotesk is used as title and Inter for contents.
- Use minimalist icons for job categories, application status, and user actions.
- A clean and responsive layout that is easy to navigate on both desktop and mobile devices.
- Subtle animations for loading states and transitions to enhance user experience.
